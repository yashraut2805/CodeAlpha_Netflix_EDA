# Netflix Data Analysis and Visualization

## Project Overview
This project is part of the CodeAlpha Data Analytics Internship.

The project focuses on Exploratory Data Analysis (EDA) and Data Visualization using the Netflix Titles Dataset. The analysis helps identify trends, patterns, and insights related to Netflix content.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset
Netflix Titles Dataset from Kaggle

## Features of Analysis
- Movies vs TV Shows comparison
- Top countries producing Netflix content
- Ratings distribution analysis
- Release year trends
- Data visualization using graphs and charts

## Project Output
The project generates visual insights using different types of plots and charts.

## Learning Outcome
Through this project, I learned:
- Data cleaning and analysis
- Exploratory Data Analysis (EDA)
- Data visualization techniques
- Using Python libraries for analytics
